# Apex-Form-SEMUSB
Formulário destinado a secretaria de segurança urbana

Tive a missão de desenvolver para prefeitura de Jõao Pessoa/PB em oracle Apex, duas aplicações.

-  1ª Formulário destinado aos candidatos que iriam concorrer às vagas da guarda municipal do município
-  2º Área de visualização dos dados para avaliação dos candidatos.
